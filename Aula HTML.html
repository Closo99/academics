Sobre o HTML

[Voltar]

HTML é uma linguagem de marcação ultilizada na construção de páginas na Web criada pelo britâncico Tim Bernes-Lee. Os navegadores (browsers) tem a capacidade de "traduzir" essa linguagem de forma eficiente.

Se compararmos páginas Web com os jornais, HTML seria todo conteudo impresso nas folhas, de forma bem enxuta, sem muito design envolvido, pois isso será por parte do CSS, mas isso é assunto para outra aula.

Ao visitar uma página Web, podemos perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa estrutura é estabelecida través do HTML.No inicio da web, era comum encontrar sites apenas contendo textos e imagens simples, com estrutura básica e sem estilizações. Porém, nos dias atuais, muito dificilmente você encontrará sites que possuam apenas elementos HTML. Portanto, podemos considerar o HTML o “esqueleto” da página.


O que são Tags

[Voltar]

Foi dito no último tópico, que HTML é uma linguagem de marcação, que estrutura páginas da Web, mas como isso realmente funciona? Essas marcações são as chamadas "Tags". As Tags são códigos que definem toda a estrutura da página , tais como seu tamanho, fonte da letra, as cores, as quebras de linha, etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fachamento. A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.


Exemplos de Tags

[Voltar]

Há muitas tags que podem ser usadas, porém nesse artigo mencionarei apenas alguns exemplos e suas funções.

<h1> (tags de título)

As tags de título variam entre seis níveis hierárquicos, e para definílos, usamos:

<h1></h1> - Título de maior valor hierárquico.
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6> - Título de menor valor hierárquico.

Os títulos vão criar a primeira estrutura organziacional básica do texto, uma vez que irá diferenciar os tipos de títulos, e separar parágrafos distintos.


<p> (parágrafo)

Essa tag é a principal tag de texto, ela irá inciar e terminar uma paragráfo, e é ultilizada da seguinte forma:

<p>texto</p>


<mark> (destaque)

Essa tag cria uma marcação no texto, semelhante a um marca-texto colorido que usamos no papel. Um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

<mark>texto</mark>


<small> (texto menor)

O elemento small agora é empregado para marcar avisos obrigatórios, restrições legais ou restrições de uso de conteúdo como copyrights, ou seja, não é apenas para tornar o tamanho do texto menor, mas na prática deixa o texto menor.

<small>texto</small>


<i> (texto italico)

Representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".

<i>texto</i>


<u> (texto sublinhado)

Essa tag seve para sublinhar o texto selecionado.

<u>texto</u>


<strong> (texto negrito)

O elemento HTML strong dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

<strong>texto</strong>


<ol> (lista ordenada)

O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

<ol>

  1. <li>texto</li>
  2. <li>texto</li>
  3. <li>texto</li>
</ol>


<ul> (lista desordenada)

O elemento HTML <ul> (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo, ou um quadrado.

<ul>

  • <li>texto</li>
  • <li>texto</li>
  • <li>texto</li>
</ul>


<li> (lemento lista)

O elemento HTML <li> (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada (<ol>), uma lista desordenada (<ul>) , ou um menu e representa uma única entidade dessa lista. Em menus e listas desordenadas a relação de itens é exibida, normalmente, usando pontos de marcação (as bolinhas). Em listas ordenadas eles são, comumente, mostrados com algum contador ascendente - como um número, ou letra - à sua esquerda.

<ul>

  • <li>texto</li>
  • <li>texto</li>
  • <li>texto</li>
</ul>


<a> (ancora ou link)

O elemento <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

<ul>

    <li><a href="https://exemplo.com">Site</a></li>
    <li><a href="exemplo@example.com">Email</a></li>
</ul>


<sub> (texto subscrito)

A tag <sub> é usada para adicionar um texto subscrito ao documento HTML. A tag <sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O para ser escrito como H2O.

<sub>texto</sub>


<sup> (texto sobrescrito)

A tag <sup> é usada para adicionar um texto sobrescrito ao documento HTML. A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé.

<sup>texto</sup>


<blockquote> (citação de bloco)

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo

<blockquote>

texto
</blockquote>


<font> (fonte)

O marcador <font></font> "Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

<font face="Verdana,Geneva,Arial">Texto</font>
<font color="Red"> Texto</font>


<del> (texto excluído)

O elemento HTML <del> (ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

<del>texto</del>


<abbr> (abreviação)

O Elemento HTML <abbr> (ou Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

Obama é presidente dos <abbr title="Estados Unidos da América">EUA</abbr>
